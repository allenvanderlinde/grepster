Creates a child process and sends it an argument list for remote commands. SpawnAndRun also records the child's stdout to grepster's console.

Parameters
[in]pathLocal directory path to external application.
[in]argsArgument list passed to child process.
Return values
std::stringReturns the process's stdout buffer as a string.

grepster can launch external applications and send them arguments as if it were a terminal.

For example:

1 C:\Users\Admin>psftp admin@127.0.0.1 -pw passwd

can be launched with grepster code as:

1 *Console << SpawnAndRun("C:\\Program Files (x86)\\PuTTY\\", "psftp.exe admin@127.0.0.1 -pw passwd");

grepster uses user-configured tools and an argument format for sending those tools remote commands for grepping and downloading operations.
